# scrimshaw

> **Archived:** This repository is no longer maintained and is preserved for reference. Feel free to fork and adapt the code under the terms of the MIT License.

Python and Grasshopper tools for Rhino, created to support geometry processing and design workflows.

## Structure

* `rhino-python/scripts/` — Python scripts for Rhino and Grasshopper.
* `grasshopper/definitions/` — Grasshopper definition files (`.ghx`).
* `grasshopper/examples/` — Example Rhino (`.3dm`) files and supporting assets.
* `assets/` — Images, animations, and other documentation resources.
